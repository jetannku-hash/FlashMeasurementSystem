# A3 幾何基元擴充 — 任務型執行計畫

> 產出日期：2026-06-25
> 範圍（依使用者決策）：**橢圓 + 圓弧(卡尺 + 擬合) + 矩形**，每個基元做**完整 GUI 接線**並**同步擴充距離/角度量測**。
> 「點(point)」改歸 **A5 幾何構造運算**（兩線/線圓交點），本計畫不含。
> 所有 HALCON operator 簽名已對 `halcon_pdf/reference/reference_hdevelop.txt` 核實（規則：圖譜=線索、reference/原碼=事實）。
> 執行慣例：本專案 feature-adapter 五件套（Domain DTO → Application 介面 → Halcon 轉接 → console 測試 → WinForms 接線），舊式 `.csproj` 需手動加 `<Compile Include=...>`。

## 0. 規格決策（已由使用者於 2026-06-25 確認，鎖定）

- **Q-E1 橢圓擬合演算法預設** → **`geotukey`**（與 circle fitter 一致、抗 outlier）。白名單仍保留全部 10 個合法值。
- **Q-R1 矩形擬合輸入來源** → **(a) `edges_sub_pix` 取輪廓 → `select_contours_xld` 挑矩形邊**。需在 adapter 處理斷裂輪廓（必要時 `union_adjacent_contours_xld` 合併、`select_contours_xld` 依長度/曲率篩四邊）。**不**走 threshold 路徑。
- **Q-AC1 圓弧卡尺 ROI 互動** → **先用「數值輸入框」起步**（中心 Row/Col、半徑、起始角、角度範圍、環寬以欄位輸入）；互動式拖拉弧 ROI 延後，不在本計畫範圍。
- **Q-D1 距離/角度新組合** → **最小集**：`EllipseCenterToX`、`RectCenterToX`、`ArcToX(視為圓)`；邊到邊用既有 `ContourMaxMin`。衍生幾何映射：橢圓→中心點、矩形→中心點、圓弧→等效圓(沿用 circle 結果)。

## 階段 A3-E：橢圓擬合（範本切片，其餘基元照此結構）

**功能**：對一段 XLD 輪廓擬合橢圓，回傳中心 (Row,Col)、主軸角 Phi、長半軸 Radius1、短半軸 Radius2、起訖角；用於長圓孔、橢圓特徵、傾斜圓投影。

**核實簽名**（reference L175603）：
```
fit_ellipse_contour_xld(Contours : : Algorithm, MaxNumPoints, MaxClosureDist,
    ClippingEndPoints, VossTabSize, Iterations, ClippingFactor
    : Row, Column, Phi, Radius1, Radius2, StartPhi, EndPhi, PointOrder)
```
- 比 circle 多一個 **VossTabSize**（僅 `voss` 演算法用，其餘填預設如 100）。
- 最少點數 = **5 + 2 × ClippingEndPoints**（reference L175690）。
- Algorithm 合法值：`fitzgibbon/fhuber/ftukey/geometric/geohuber/geotukey/voss/focpoints/fphuber/fptukey`。

  - 新增 `Domain/EllipseFitting/EllipseFittingResult.cs`：`CenterRow/CenterColumn/Phi/Radius1Px/Radius2Px/StartPhi/EndPhi/PointOrder/UsedPoints/ResidualRms/Success/ErrorMessage`。
  - `.csproj`(Domain) 加兩筆 `<Compile Include>`。

## 階段 A3-R：矩形擬合

**功能**：擬合旋轉矩形，回傳中心、主軸角 Phi、半邊長 Length1/Length2；用於量方孔、晶片外框、鍵槽。

**核實簽名**（reference L175931）：
```
fit_rectangle2_contour_xld(Contours : : Algorithm, MaxNumPoints, MaxClosureDist,
    ClippingEndPoints, Iterations, ClippingFactor
    : Row, Column, Phi, Length1, Length2, PointOrder)
```
- Algorithm 合法值：`regression/huber/tukey`（**與 circle/ellipse 不同組**，白名單要分開）。
- Length1/Length2 是**半邊長**；Phi 為 Length1 邊與水平軸夾角（逆時針，弧度）。

### 任務清單
- [ ] **R0.（Q-R1 已定：edges_sub_pix + select_contours_xld）** 設計輪廓挑選策略：對 ROI `edges_sub_pix` → 視情況 `union_adjacent_contours_xld` 接合斷裂 → `select_contours_xld` 依長度/方向挑出矩形四邊 → 餵 `fit_rectangle2_contour_xld`。
- [ ] **R1. Domain DTO**：`RectangleFittingParameters`(Algorithm 預設 `tukey`、MaxNumPoints=-1、MaxClosureDist=0、ClippingEndPoints=0、Iterations=3、ClippingFactor=2.0) + `IsSupportedAlgorithm`(僅三值) + `RectangleFittingResult`(Center/Phi/Length1Px/Length2Px/PointOrder/...)。`.csproj` 加 Compile。
- [ ] **R2. Application**：`IRectangleFitter.FitRectangle(...)`。
- [ ] **R3. Halcon**：`HalconRectangleFitter`，依 Q-R1 決定的輸入路徑取得四邊輪廓 → `FitRectangle2ContourXld` → map 回。
- [ ] **R4. 測試**：合成矩形點集擬合 + 邊界條件，串入 `Main()`。
- [ ] **R5. WinForms**：按鈕 + 結果 + overlay(`gen_rectangle2_contour_xld` 畫框)；`ToolType="rectangle"`。
- [ ] **驗收**：同 E 階段。

**功能**：對部分圓弧擬合出圓心/半徑/起訖角，量 R 角、扇形、圓弧段。**現有 `fit_circle_contour_xld` 已回傳 StartPhi/EndPhi**（見 `HalconCircleFitter` L66-67），所以擬合本身已具備，本階段主要補「弧」語意與顯示。

- [ ] **驗收**：GUI 對一段弧擬合，疊加只畫弧段不畫整圓。

## 階段 A3-AC：圓弧卡尺（gen_measure_arc）

**功能**：沿環形弧佈設量測線、抓垂直於弧的邊（弧上的卡尺）。主流量測儀量「圓周上等分特徵、齒、孔位」用此。屬 1D measuring，擴充現有 EdgeDetector 路徑。

**核實簽名**（reference L3304）：
```
gen_measure_arc( : : CenterRow, CenterCol, Radius, AngleStart, AngleExtent,
    AnnulusRadius, Width, Height, Interpolation : MeasureHandle)
```
- 之後接 `measure_pos`/`measure_pairs`（與現有 `RunMeasurePos` 同一套後處理）。`AngleExtent>0` 逆時針。`AnnulusRadius` = 環寬一半。

### 任務清單
- [ ] **AC0.（Q-AC1 已定：數值輸入框起步）** GUI 用欄位輸入弧參數，不做互動拖拉。
- [ ] **AC1. Domain**：新增 `ArcMeasureRoi`(CenterRow/Col、Radius、AngleStart、AngleExtent、AnnulusRadius) + 驗證(半徑>0、環寬>0)。
- [ ] **AC2. Halcon**：在 `HalconEdgeDetector` 加 `DetectEdgesOnArc(...)` 或新 adapter：`GenMeasureArc` → `MeasurePos`，重用既有 edge 後處理與單通道轉換、`CloseMeasure` 釋放。沿用現有 clamp/log/例外風格。
- [ ] **AC3. WinForms**：依 Q-AC1，先做數值輸入面板或互動弧 ROI；overlay 畫弧形量測帶 + 抓到的邊點。
- [ ] **AC4. 測試**：合成環形邊影像（或既有測試圖）驗證抓到預期邊數。
- [ ] **驗收**：GUI 對圓周特徵量測，邊點落在弧上正確位置。

**現況**：`DistanceMeasurementType` = `PointToPoint/PointToLine/LineToLine/CircleToCircle/ContourMaxMin`；`MeasurementTool.RefToolIds` 已支援複合工具參考元素工具。

### 任務清單

- 每階段獨立可交付、可 rollback；不混入無關重構（global/project CLAUDE.md）。
