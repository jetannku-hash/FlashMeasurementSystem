### 四、Halcon 17.12 API 相容性修正

開發手冊範例使用了 Halcon 18+ 的 OOP API，在 17.12 環境中需調整：

| 手冊寫法（18+） | 實際使用（17.12） |
|-----------------|-------------------|
| `HShapeModel.CreateShapeModel(...)` (10 參數) | `HOperatorSet.CreateShapeModel(...)` (procedural, 9+out HTuple) |
| `HTuple.TupleRad(angle)` (static) | `new HTuple(angle * PI / 180)` (手動轉換) |
| `HOperatorSet.ReduceDomain(..., out HImage)` | `HOperatorSet.ReduceDomain(..., out HObject)` |
| `new HShapeModel(modelID)` | `HOperatorSet.WriteShapeModel` / `ReadShapeModel` |
| `HMouseEventArgs` (Halcon 事件) | `MouseEventArgs` (標準 WinForms 事件) — 滾輪穩定性修正 |
