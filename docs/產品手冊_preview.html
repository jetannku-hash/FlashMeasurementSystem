量測功能矩陣

從擷取到報表，一條管線完成。以下為目前已實作的量測能力總覽。

01 影像前處理

影像品質檢查 + 模板匹配

亮度/對比/雜訊自動檢查；基於形狀模型的工件定位與剛體座標轉換，ROI 可隨工件平移旋轉。

IQC pass/fail · find_shape_model · vector_angle_to_rigid

1D 卡尺 + 亞像素邊緣

支援 measure_pos（矩形卡尺）與 edges_sub_pix（Canny 亞像素）。弧形卡尺 gen_measure_arc 可沿圓弧佈設量測線。

Sigma / Threshold / Polarity · 弧形 ROI 互動拖曳編輯

03 XLD 擬合

線 · 圓 · 橢圓 · 矩形

對亞像素邊緣點雲做穩健最小平方擬合。線支援封閉厚特徵 PCA 回退；圓支援整圓/弧段判別（StartPhi/EndPhi）。

fit_line/circle/ellipse/rectangle2_contour_xld
[ 截圖：四種擬合結果並排 ]
04 尺寸量測

距離 · 角度

點到點 / 點到線 / 線到線 / 圓到圓距離；兩線夾角 AcuteAngle[0,90°]。可組合 line、circle、以及構造工具的輸出。

mm / deg · 公差雙邊判定 Nominal±Lower/Upper
05 幾何構造

交點 · 中線 · 投影

兩線交點（平行偵測）、對稱中線/角平分線、點投影到線。純 Domain 數學，產出可被距離/角度複合工具引用。

GeometricPrimitive {Point|Line|Circle} · Resolver
06 形位公差（GD&T v1）

真圓度 · 真直度 · 平行度 · 垂直度 · 同心度

單邊判定 0 ≤ 偏差 ≤ 公差帶 T。真圓度用 max−min 徑向（真值）；平行/垂直用 L·sinΔθ；同心 2×圓心距。單一基準。

OK dev≤T · NG dev>T · 僅上限警示
07 配方系統

Recipe Editor · 一鍵量測

以 .zcp（JSON）持久化多工具配方。編輯器非模態執行、與主視窗共用影像並即時取 ROI。一鍵流程：IQC→匹配→量測→判定→CSV。

Schema v5 · Newtonsoft · 向後相容 · 原子寫入
08 校正 · 報表

像素→mm 校正 + CSV 輸出

雙點像素尺寸校正（CalibrationProfile 持久化）。量測結果自動輸出 CSV，含時間戳、工具名、偏差、判定、訊息。

pixelSizeUmX/Y · 單一 profile 配方參考 · fallback 防鎖
